# 🧹 Chore Quest Hub

A fun, AI-powered chore tracking app where working hard earns you epic breaks — complete with a Game Hub, an AI buddy, and a chatbot!

## ✨ Features
- ✅ Chore Tracker — add, complete, and earn break time
- ⏸️ Break Timer — timed breaks that launch the Game Hub
- 🎮 Game Hub — 2D + 3D games to play on your break
- 🤖 AI Buddy — plays games for you or gets you started
- 💬 Chatbot — chat companion + create new games
- 🔁 Seamless flow — chores → break → games → back to chores
